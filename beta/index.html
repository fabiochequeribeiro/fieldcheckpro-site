Como funciona

Um processo simples para testar com segurança.

O cliente baixa o APK, recebe login de teste, usa por 30 dias e envia devolutivas para evoluirmos com base no uso real.

1

Solicitar acesso

A empresa informa dados básicos para criarmos o usuário, liberar módulos e definir o período beta.

2

Baixar APK

O APK fica hospedado no Drive e disponível nesta página para instalação direta no Android.

3

Entrar no Hub

Gestores acessam o FieldCheck Hub para acompanhar dados, usuários, feedbacks e status do teste.

4

Enviar devolutiva

Ao final, coletamos comentários, notas, melhorias e resultados para aprimorar o produto.

O que o cliente testa

App, Portal e IA assistida no mesmo programa.

  • Checklists digitais com fotos e observações
  • Assinaturas e relatórios PDF
  • Histórico e status de sincronização
  • Assistente IA para sugestões e resumos
  • Portal FieldCheck Hub para gestores
  • Feedback para melhorias do produto

Instalação Android

Orientação para instalar o APK fora da Play Store.

Enquanto o FieldCheck Pro Beta não estiver publicado na Play Store, o Android pode solicitar permissão para instalar aplicativos de fontes externas.

✓

Baixe o APK

Clique no botão de download e aguarde o arquivo terminar de baixar.

✓

Autorize a instalação

Se o Android pedir, permita instalação pelo navegador ou gerenciador de arquivos.

✓

Faça login

Use o usuário e senha liberados para a empresa participante do beta.

✓

Envie feedback

Registre dificuldades, sugestões e resultados para a evolução da versão 2.0.

Pronto para testar?

Comece com 30 dias e prorrogue se precisar de mais tempo.

Depois do período inicial, podemos liberar +15 ou +30 dias para aprofundar os testes e medir resultados.
